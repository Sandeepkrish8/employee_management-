# Employee Management Form

A simple employee management system that allows you to add and delete employees with form validation.

## Features

- ✅ Add employees with name, profession, and age
- ✅ Automatic unique ID generation
- ✅ Form validation with error messages
- ✅ Success confirmation messages
- ✅ Delete employees functionality
- ✅ Responsive UI design

## How to Use

1. Open `index.html` in your web browser
2. Fill in the employee details (name, profession, age)
3. Click "Add Employee" to add the employee
4. View added employees in the list below
5. Click "Delete User" to remove an employee

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Validation Rules

- All fields are required
- Error message shown in red if any field is empty
- Success message shown in green when employee is added successfully
